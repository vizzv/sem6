by default in js when you compare any values it is providing boolean results.

number object:
the number object is an object wrapper for primitive numeric values.
the number objects are created with new Number()

properties:
MAX_VALUE=it returns largest number possible in javascript.
MIN_VALUE=it returns smallest number possible in javascript.

        if input is number it returns true otherwise false.
        syntax: isFinite(number)  
    3->isNaN():
        this is exactlly opossite of  isFinite().
        synatax: isNaN(number)
